## Project Summary

This project involves the complete design and assembly of a Shaper Tool Head using SOLIDWORKS 2022. The assembly consists of 16 precision-engineered components, including the vertical plate, swivel plate, tool holder, screw mechanism, handle assembly, pivot system, and fastening elements. The project demonstrates mechanical assembly design, engineering drawings, material selection, and manufacturing documentation.

## Project Overview

The Shaper Tool Head is a critical component of a conventional shaper machine, responsible for securely holding and accurately positioning the cutting tool during machining operations. It provides controlled angular adjustment and rigid clamping, enabling precise cutting and improved machining flexibility.

## Working Principle

The Shaper Tool Head securely holds the cutting tool while allowing angular adjustment through the swivel mechanism. The tool position can be adjusted and locked using the clamping screw arrangement, enabling the machining of flat, inclined, and angular surfaces.

The combination of the swivel plate, pivot pin, tool holder, and clamping mechanism ensures rigidity, accuracy, and smooth adjustment during machining operations.

## Project Development Process

### Concept Development

Studied the construction and working mechanism of a conventional shaper tool head.

### CAD Modeling

Designed all individual components including the vertical plate, swivel plate, tool holder, screw mechanism, and handle assembly.

### Assembly Creation

Integrated all components into a fully constrained assembly using SOLIDWORKS mates and alignment constraints.

### Verification

Checked assembly fitment, interference conditions, rotational movement, and component alignment.

### Documentation

Prepared assembly drawings, exploded views, detailed manufacturing drawings, and Bill of Materials (BOM).
